# Clinic Booking Frontend

Frontend cho hệ thống đặt lịch khám bệnh.

## Công nghệ sử dụng
- React + TypeScript
- Vite
- Axios
- React Router
- CSS Modules

## Chức năng chính
- Đăng ký / Đăng nhập
- Tìm kiếm bác sĩ theo chuyên khoa
- Đặt lịch khám bệnh
- Quản lý lịch hẹn (Bệnh nhân / Bác sĩ)
- Trang quản trị (Admin)
